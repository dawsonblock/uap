# UAP: Evidence, Physics, and the Limits of Speculation

This repository contains a disciplined manuscript bundle about unidentified anomalous phenomena (UAP), public evidence, and the limits of speculative physics.

The central position is deliberately narrow:

- Credible anomaly does not equal verified origin.
- Classified allegation does not equal evidence.
- Physics vocabulary does not equal engineering reality.

The canonical manuscript is `uap-comprehensive.md`. The other files are supporting modules, appendices, or source drafts.

## Publication rules

Treat `uap-comprehensive.md` as the only canonical version unless a future release says otherwise. Speculative sections are not proposed mechanisms; they are conditional maps of what would have to be true if extreme performance were independently verified and conventional explanations failed.

Before publication, verify every official date, case title, action number, report title, and source URL against the original source. Small factual defects damage this argument more than rhetorical weakness does.
